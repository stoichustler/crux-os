These web pages are served by a port of THTTPD running on top of ROUX. ROUX includes an embedded TCP/IP state that derives from the uIP.

Click on the links above to exercise THTTPD's CGI capability under ROUX. Depending on the configuration, clicking the links will execute the CGI program from either:

  1. An NXFLAT program residing in a ROUX ROMFS file system, or
  2. A built-in application managed by a ROUX BINFS file system.
